## 一、动态场景描述系统

### 1.1 设计目标

- 根据玩家背包物品、已完成支线、游戏内时间（昼夜/月相）等状态，**动态生成**场景描述文本。
- 避免硬编码多个版本的场景，使用**条件拼接**机制。
- 描述变化实时反映，增强“世界因我而变”的沉浸感。

### 1.2 数据结构

在 `gameStore.js` 中已存在的 `run` 状态外，增加游戏内时间概念（可选）：

```javascript
// gameStore.js state
run: {
  // ... existing
  game_time: {        // 游戏内时间（可选，用于环境变化）
    hour: 20,         // 20点（黄昏）
    day: 1,
    phase: 'dusk'     // 'dawn', 'day', 'dusk', 'night'
  }
}
```

### 1.3 动态描述生成器

创建 `src/utils/dynamicDesc.js`：

```javascript
import { useGameStore } from '@/store/gameStore';

/**
 * 动态场景描述生成器
 * 每个房间注册一个描述函数，根据状态返回不同段落
 */
export const dynamicDescriptions = {

  // 画室动态描述
  studio_entry: (store) => {
    const descParts = [];
    
    // 基础描述
    descParts.push(`画室位于庄园二层东侧，是一间宽敞明亮的房间。月光透过天窗洒落，照亮了满墙的画作。`);
    
    // 根据支线进度添加细节
    if (store.getFlag('side_painting_completed')) {
      descParts.push(`七幅画作已经完成，色彩鲜艳，仿佛有生命在其中流动。`);
    } else if (store.getFlag('side_painting_triggered')) {
      descParts.push(`七幅画作已经完成，但空气中似乎弥漫着一丝忧伤。`);
    } else {
      descParts.push(`画框上标着不同的颜色名称：赤、橙、黄、绿、青、蓝、紫。但画布都是空白的。`);
    }
    
    // 根据物品添加细节
    if (store.hasItem('银手镯')) {
      descParts.push(`你手腕上的银手镯在月光下微微发光，画中的女子似乎正看着那道银光。`);
    }
    if (store.hasItem('伊莲娜纪念徽章')) {
      descParts.push(`那枚心形徽章在你胸口轻轻震动，仿佛在回应画室里的某种呼唤。`);
    }
    
    // 根据时间添加环境变化
    if (store.run.game_time?.phase === 'night') {
      descParts.push(`月光如水，彩色玻璃窗在地板上投下七色光斑。`);
    } else {
      descParts.push(`阳光透过彩色玻璃窗，在地板上投下斑斓的光影。`);
    }
    
    // 根据好感度添加角色情绪
    const elenaAff = store.profile.character_affection?.elena || 0;
    if (elenaAff >= 5 && store.getFlag('side_painting_completed')) {
      descParts.push(`你隐约听到一声轻柔的叹息，像是有人在说：“谢谢你。”`);
    }
    
    return descParts.join('\n\n');
  },

  // 大厅动态描述
  hall_main: (store) => {
    const descParts = [];
    descParts.push(`你站在大厅中央。大厅两侧各立着四座大理石雕像。`);
    
    const medalCount = store.run.medals.length;
    if (medalCount === 0) {
      descParts.push(`壁炉里有烧焦的纸片。通往其他房间的走廊隐约可见。`);
    } else if (medalCount < 7) {
      descParts.push(`壁炉台上摆放着 ${medalCount} 枚徽章，它们散发着柔和的光芒。墙上七幅抽象画中，已有 ${medalCount} 幅显现出清晰的图案。`);
    } else {
      descParts.push(`七枚徽章齐聚，大厅中央的石台发出轰鸣，露出一条螺旋石梯。`);
    }
    
    // 根据管家支线进度
    if (store.getFlag('side_butler_completed')) {
      descParts.push(`管家奥尔德斯站在阴影中，他的眼中似乎多了一丝释然。`);
    } else if (store.getFlag('side_butler_triggered')) {
      descParts.push(`管家不在大厅，壁炉台上压着一张纸条。`);
    }
    
    // 根据时间
    if (store.run.game_time?.phase === 'night') {
      descParts.push(`月光透过高处的彩窗，在大理石地板上投下冷白色的光。`);
    }
    
    return descParts.join('\n\n');
  },

  // 音乐室动态描述
  musicroom_entry: (store) => {
    const descParts = [];
    descParts.push(`音乐室位于庄园一层西侧，穹顶圆形大厅。一架巨大的管风琴矗立在中央。`);
    
    if (store.getFlag('side_music_completed')) {
      descParts.push(`空气中似乎还回荡着埃莉诺的第七交响曲，余音缭绕。`);
    } else if (store.hasItem('埃莉诺的琴弓')) {
      descParts.push(`你手中的琴弓微微颤动，仿佛在催促你演奏什么。`);
    }
    
    if (store.hasItem('共鸣水晶')) {
      descParts.push(`你背包里的共鸣水晶轻轻震动，与管风琴的音管产生共鸣。`);
    }
    
    return descParts.join('\n\n');
  },

  // 地下室动态描述
  basement_entry: (store) => {
    const descParts = [];
    descParts.push(`地下室阴暗潮湿，空气中夹杂着铁锈与泥土的气味。`);
    
    if (store.getFlag('side_underground_completed')) {
      descParts.push(`洞穴入口已被封住，地下的回响已然沉寂。`);
    } else if (store.getFlag('side_underground_triggered')) {
      descParts.push(`墙壁上的裂缝中透出冷风，隐约可以听到敲击声。`);
    }
    
    if (store.hasItem('符文石')) {
      descParts.push(`你手中的符文石微微发热，似乎在指引着什么。`);
    }
    
    return descParts.join('\n\n');
  }
};
```

### 1.4 在场景模块中使用动态描述

修改房间入口场景（如 `studio_entry`），将静态 `desc` 替换为函数：

```javascript
// studio.js
import { dynamicDescriptions } from '@/utils/dynamicDesc';

export default {
  'studio_entry': {
    desc: (store) => dynamicDescriptions.studio_entry(store),
    // 其他属性不变
    options: [ ... ]
  }
}
```

**注意**：需要将 `store` 实例传递给场景的 `desc` 函数。在 `GameView.vue` 的 `loadScene` 中，调用 `desc` 时传入 `store`：

```javascript
// GameView.vue
const sceneDesc = typeof scene.desc === 'function' ? scene.desc(gameStore) : scene.desc;
```

### 1.5 游戏内时间推进（可选）

在每次场景切换或选择选项时，推进时间：

```javascript
// gameStore.js actions
advanceTime(minutes = 15) {
  let newHour = this.run.game_time.hour + minutes / 60;
  if (newHour >= 24) {
    newHour -= 24;
    this.run.game_time.day++;
  }
  this.run.game_time.hour = newHour;
  if (newHour >= 18 || newHour < 6) {
    this.run.game_time.phase = 'night';
  } else if (newHour >= 6 && newHour < 12) {
    this.run.game_time.phase = 'dawn';
  } else if (newHour >= 12 && newHour < 18) {
    this.run.game_time.phase = 'day';
  } else {
    this.run.game_time.phase = 'dusk';
  }
  this.saveState();
}
```

在每次选择选项时调用 `store.advanceTime()`。

## 二、隐藏的“观察”交互系统

### 2.1 设计目标

- 在每个房间增加一个独立的“🔍 仔细观察”选项。
- 点击后显示该房间的额外环境细节（不直接给解谜答案，但提供氛围或隐藏伏笔）。
- 某些观察点可能解锁隐藏线索或触发小事件。

### 2.2 数据结构

在 `run` 中记录已观察过的点，避免重复显示：

```javascript
run: {

  observed_details: []  // 已观察过的细节ID
```

### 2.4 在场景选项中动态添加“观察”选项

修改 `GameView.vue` 的选项渲染逻辑，为每个场景动态添加“仔细观察”选项（如果尚未观察过所有点）：

```javascript
// GameView.vue 的 loadScene 或 computed
const enhancedOptions = computed(() => {
  const baseOpts = currentOptions.value;
  const sceneId = currentSceneId.value;
  const points = observationPoints[sceneId] || [];
  const observedIds = gameStore.run.observed_details || [];
  const hasUnobserved = points.some(p => !observedIds.includes(p.id));
  
  if (hasUnobserved && !baseOpts.some(opt => opt.text === '🔍 仔细观察')) {
    return [
      ...baseOpts,
      { text: '🔍 仔细观察', target: 'system_observe', isObservation: true }
    ];
  }
  return baseOpts;
});
```

### 2.5 观察场景处理

在 `transitions.js` 或单独模块中定义 `system_observe` 场景：

```javascript
// transitions.js
'system_observe': {
  desc: (store) => {
    const sceneId = store.run.current_scene_id;
    const points = observationPoints[sceneId] || [];
    const observedIds = store.run.observed_details || [];
    const unobserved = points.filter(p => !observedIds.includes(p.id));
    
    if (unobserved.length === 0) {
      return '你仔细环顾四周，但没有发现任何新的细节。';
    }
    
    // 随机或按顺序展示一个未观察的点
    const point = unobserved[0];
    // 记录已观察
    store.run.observed_details.push(point.id);
    store.saveState();
    
    // 执行奖励
    if (point.reward) point.reward();
    
    let msg = point.text;
    if (point.hint) msg += `\n\n（${point.hint}）`;
    return msg;
  },
  options: [
    { text: '继续探索', target: (store) => store.run.current_scene_id }
  ]
}
```

### 2.6 在注册表中注册观察场景

```javascript
  // ...
  'system_observe': () => import('./core/transitions.js')
```

## 三、日记/信件系统升级（虚拟笔记本）

- 玩家收集的日记、信件、便条以**可翻页的笔记本**形式展示。

### 3.4 虚拟笔记本组件（VirtualNotebook.vue）

创建 `src/components/VirtualNotebook.vue`：

```vue
<template>
  <div class="notebook-overlay" @click.self="close">
    <div class="notebook-container">
      <div class="notebook-header">
        <h2>{{ currentEntry.title }}</h2>
        <div class="notebook-meta">
          <span>作者：{{ currentEntry.author }}</span>
          <span>日期：{{ currentEntry.date }}</span>
        </div>
        <button class="close-btn" @click="close">✕</button>
      </div>
      
      <div class="notebook-body">
        <!-- 翻页效果 -->
        <div class="book-pages">
          <div class="page left-page" v-if="currentPage > 0">
            <div class="page-content" v-html="formattedContent(currentPage - 1)"></div>
          </div>
          <div class="page right-page">
            <div class="page-content" v-html="formattedContent(currentPage)"></div>
          </div>
        </div>
        
        <!-- 注释区域 -->
        <div class="annotation-area">
          <label>你的笔记：</label>
          <textarea 
            v-model="currentNote" 
            @blur="saveNote"
            placeholder="点击此处添加你的想法..."
            class="annotation-input"
          ></textarea>
        </div>
        
        <!-- 翻页控制 -->
        <div class="page-controls">
          <button @click="prevPage" :disabled="currentPage === 0">◀ 上一页</button>
          <span>第 {{ currentPage + 1 }} / {{ totalPages }} 页</span>
          <button @click="nextPage" :disabled="currentPage === totalPages - 1">下一页 ▶</button>
        </div>
      </div>
      
      <div class="notebook-footer">
        <button @click="close" class="close-footer-btn">关闭笔记本</button>
      </div>
    </div>
  </div>
</template>

<script setup>
import { ref, computed, watch } from 'vue';
import { useGameStore } from '@/store/gameStore';
import { journalEntries } from '@/data/journalEntries';

const props = defineProps({
  entryId: {
    type: String,
    required: true
  }
});

const emit = defineEmits(['close']);
const store = useGameStore();

const currentEntry = computed(() => journalEntries[props.entryId]);
const totalPages = computed(() => currentEntry.value?.pages.length || 0);
const currentPage = ref(0);
const currentNote = ref('');

// 加载已有的注释
watch(() => props.entryId, () => {
  const entry = store.profile.journal_entries?.find(e => e.id === props.entryId);
  if (entry && entry.user_note) {
    currentNote.value = entry.user_note;
  } else {
    currentNote.value = '';
  }
  currentPage.value = 0;
}, { immediate: true });

const formattedContent = (pageIndex) => {
  const page = currentEntry.value.pages[pageIndex];
  if (!page) return '';
  // 简单支持换行
  return page.content.replace(/\n/g, '<br>');
};

const saveNote = () => {
  const entry = store.profile.journal_entries?.find(e => e.id === props.entryId);
  if (entry) {
    entry.user_note = currentNote.value;
  } else {
    store.profile.journal_entries.push({
      id: props.entryId,
      title: currentEntry.value.title,
      content: currentEntry.value.pages[0].content,
      category: currentEntry.value.category,
      acquired_at: new Date().toISOString(),
      user_note: currentNote.value,
      is_read: true
    });
  }
  store.saveState();
};

const prevPage = () => {
  if (currentPage.value > 0) currentPage.value--;
};
const nextPage = () => {
  if (currentPage.value < totalPages.value - 1) currentPage.value++;
};
const close = () => {
  saveNote();
  emit('close');
};
</script>
```

### 3.5 在游戏中打开笔记本

修改 `InventoryPanel.vue` 或添加新按钮，当玩家点击某条日记时：

```vue
<!-- InventoryPanel.vue 中 -->
<div 
  v-for="entry in journalEntries" 
  :key="entry.id"
  class="journal-item"
  @click="openNotebook(entry.id)"
>
  📖 {{ entry.title }}
</div>
```

在父组件中处理打开：

```javascript
const openNotebook = (entryId) => {
  uiState.notebook = true;
  currentNotebookEntry.value = entryId;
};
```

在 `GameView.vue` 中添加笔记本组件：

```vue
<VirtualNotebook 
  v-if="uiState.notebook"
  :entry-id="currentNotebookEntry"
  @close="uiState.notebook = false"
/>
```
