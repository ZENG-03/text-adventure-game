# 游戏文本完整性检查与补充

根据已生成的内容，我梳理了构成一个完整文字冒险游戏所需的所有文本模块，并指出已具备和仍缺失的部分。以下将按照游戏流程，逐项补充缺失的文本。

---

## 一、已有文本模块清单

| 模块 | 包含内容 | 状态 |
|------|----------|------|
| 序章 | 午夜来信、大厅初遇 | ✅ 完整 |
| 七谜房间 | 每个房间有入口、分支、成功、失败、联动等场景，总计约3.5万字 | ✅ 完整 |
| 四个支线 | 每个支线约1万字，含触发、探索、结局 | ✅ 完整 |
| 最终密室 | 四重考验、多种结局 | ✅ 完整 |
| 成就系统 | 25个成就描述 | ✅ 完整 |

## 二、缺失文本模块

### 1. 大厅的阶段性描述
随着谜题进展，大厅应反映变化（如油画上的徽章增加、管家的态度变化）。需要为每个徽章收集阶段编写不同的描述。

### 2. 房间完成后的返回过渡
每次获得徽章后返回大厅的简短文本，以及管家可能的新对话。

### 3. 支线触发提示
当条件满足时，系统自动触发的引导文本（如管家留下纸条、画室出现异响等）。

### 4. 全局道具清单
玩家获得的每个关键道具的说明文本（可查看背包时显示）。

### 5. 死亡结局统一格式
所有死亡结局应统一风格，并包含“重新尝试”或“返回大厅”的选项。

### 6. 最终密室前的汇合
集齐七枚徽章后，从卧室到中央密室的过渡描述。

### 7. 游戏开始时的角色背景
侦探笔记的用途说明，以及游戏的基本操作提示。

### 8. 结局后的彩蛋与重玩选项
结局后应提供“重新开始”或“查看成就”的选项。

## 三、补充文本

### 3.1 大厅的阶段性描述（多状态）

**初始大厅**（已有）：
> 门厅内，水晶吊灯蒙着薄灰，墙壁上挂着七幅抽象画，每一幅都似乎在注视着你。管家奥尔德斯无声地出现……

**获得1枚徽章后**：
> 你回到大厅，将蓝宝石徽章放在壁炉台上。管家瞥了一眼，微微点头：“第一枚。还有六道门等待着你。” 墙上的第一幅抽象画似乎变得明亮了一些。

**获得3枚徽章后**：
> 大厅的光线似乎比之前更亮了。三枚徽章在壁炉台上并排闪烁，墙上的三幅抽象画已经焕发出色彩。管家递给你一杯热茶：“你比我想象的更快。但后面的谜题会更难。”

**获得6枚徽章后**：
> 大厅几乎变了一副模样。六枚徽章的光芒照亮了整个房间，墙上的六幅抽象画已经变成了生动的场景——图书馆、钟楼、音乐室……只差最后一幅。管家的声音里带着一丝期待：“最后一枚徽章在卧室。主人的私人空间，藏着最终的秘密。”

**集齐7枚徽章后**：
> 七枚徽章齐聚的瞬间，大厅中央的石台发出轰鸣，缓缓下沉，露出一条螺旋石梯。管家站在阴影中，深深鞠躬：“主人等候已久。请沿着这条路，去揭开最后的谜底。”

### 3.2 房间完成后的返回过渡（通用模板）

每次获得徽章后，自动返回大厅，并显示以下文本（可根据房间稍作变化）：

> 你将徽章收入怀中，最后看了一眼[房间名称]。似乎有什么东西在角落里轻轻叹息，然后一切归于沉寂。当你走出房门时，门在你身后无声地关闭，门把手上的光泽消失了——仿佛它已经完成了自己的使命。你回到大厅，将[颜色]徽章与其他徽章并排放在壁炉台上。管家无声地递上一杯热茶，没有多问。

### 3.3 支线触发提示

**触发管家支线（完成任意三个房间后，返回大厅时）**：
> 你走进大厅，发现管家不在往常的位置。壁炉台上压着一张纸条：“我去地下室取些东西，请稍候。” 你拿起纸条，背面隐约有字迹，似乎是被人擦过的痕迹：“不要相信……” 你的直觉告诉你，这位沉默寡言的管家藏着秘密。

**触发画室支线（完成画室谜题后，再次进入画室时）**：
> 画室的门虚掩着。你推门而入，发现南墙的彩色玻璃窗投下的光斑在地面上拼出一个女人的侧影轮廓。七幅完成的画作中，似乎每一幅里都有一个女子的身影若隐若现。你的手不自觉地抬起，触碰赤色画的表面——画面微微发烫，一个声音在脑海中响起：“他把我画进了每一道光里，却始终画不出我的灵魂。”

**触发地下支线（完成地下室谜题后，且至少完成两个其他房间，再次经过地下室入口时）**：
> 你经过地下室的入口，注意到旁边的墙壁上有一道之前没见过的裂缝。裂缝很细，但有冷风从里面渗出。你将耳朵贴近，隐约听到有节奏的敲击声——像是有人在用石头敲打岩石，一下，两下，三下，停顿，重复。你询问管家，他的眼神闪烁：“那只是岩壁。” 你知道他在说谎。

**触发音乐室支线（完成音乐室谜题后，再次经过音乐室时）**：
> 当你经过音乐室时，紧闭的门内传出微弱的音乐声——不是管风琴的庄严旋律，而是一把小提琴的独奏，曲调忧伤而优美。你推开门，音乐声戛然而止，空无一人。小提琴的琴弦还有余温，琴谱架上多了一份你从未见过的手稿：《第七交响曲：未完成》。

### 3.5 死亡结局统一格式

所有死亡结局均采用以下模板，并在末尾提供选项：

> 你的意识逐渐模糊。在最后的瞬间，你听见管家奥尔德斯的声音从遥远的地方传来：“鲁莽是解谜的大敌。” 世界陷入黑暗。
> 
> **选项：**
> - 从最近的存档点重新尝试 [重新开始当前房间]
> - 回到大厅，暂时放弃这个房间 [返回大厅]
> - 结束游戏 [退出]

**注意**：为了降低挫败感，死亡结局不会永久终止游戏，而是允许从房间入口或大厅重试。

### 3.6 最终密室前的汇合

集齐七枚徽章后，从卧室出发前往中央密室的过渡文本：

> 当第七枚徽章嵌入油画的瞬间，整个卧室都开始震动。油画中的七扇窗户同时亮起，光芒汇聚成一束，射向地面。地板裂开，露出一条螺旋石梯，石阶向下延伸，消失在黑暗中。
> 
> 管家不知何时站在门口，他的眼中映着七色光：“这条路通往庄园的最深处，也是主人最后的谜题所在。我在这里等你回来。”
> 
> 你深吸一口气，踏上了石阶。身后的光芒逐渐暗淡，前方的黑暗却透出微弱的荧光。石梯似乎很长，每一步都回荡着你的心跳。不知走了多久，你终于看到一扇敞开的石门——中央密室到了。

### 3.7 游戏开始时的角色背景

在序章前插入一段，介绍玩家角色和侦探笔记：

> 你是私家侦探塞拉斯·诺斯，以破解冷案和谜团闻名。你从不依靠运气，只相信逻辑和观察。随身携带的侦探笔记记录了你经手的每一个案件，空白页正等待着新的谜题。
> 
> 此刻，一封午夜来信将你引向迷雾山谷中的谜语馆。你合上笔记，将地图折好放进口袋。直觉告诉你，这次委托将不同于以往任何案件。

**侦探笔记的用途**：在游戏过程中，所有获得的关键线索、日记内容、地图都会自动记录在笔记中。玩家可以随时“查看笔记”来回顾已获得的线索。

### 3.8 结局后的彩蛋与重玩选项

每个结局后统一添加以下内容：

> **—— 游戏结束 ——**
> 
> 感谢你体验《谜语遗产：七重谜域》。你解锁的结局：[结局名称]
> 
> 已解锁成就：[列出本次游玩解锁的成就]
> 
> **选项：**
> - 重新开始游戏 [重置所有进度]
> - 查看成就列表 [显示已解锁成就]
> - 退出游戏 [返回标题]

**特殊彩蛋**：在“七重谜语的真相”结局后，额外显示：

> 在制作人员名单滚动时，你注意到最后一行小字：“献给所有保持好奇心的探索者。” 画面淡出，庄园的七扇窗户依次熄灭。但你知道，当下一个满月升起时，会有人再次推开那扇门。

## 四、需要代码实现的状态变量

为确保所有分支和支线正确触发，游戏需要记录以下状态：

| 变量名 | 类型 | 说明 |
|--------|------|------|
| `徽章列表` | array | 已获得的徽章（7个布尔值） |
| `房间完成状态` | array | 每个房间是否完成 |
| `支线触发状态` | array | 四个支线是否已触发 |
| `支线完成状态` | array | 四个支线是否已完成 |
| `支线结局选择` | array | 每个支线中选择的路线 |
| `道具列表` | array | 玩家持有的所有关键道具 |
| `线索列表` | array | 已阅读的日记、信件等 |
| `大厅徽章计数` | int | 用于大厅描述变化 |
| `管家对话索引` | int | 管家的不同阶段对话 |
| `最终抉择` | string | 玩家在最终密室的选择 |
